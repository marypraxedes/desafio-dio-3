# 🧙‍♂️ Classificador de Nível de Herói

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação utilizando JavaScript.

A aplicação realiza a classificação de um herói com base na quantidade de experiência (XP), utilizando estruturas condicionais para determinar o nível correspondente.

---

## ⚙️ Como executar

1. Clone este repositório:
git clone https://github.com/marypraxedes/desafio-dio-3.git

2. Acesse a pasta do projeto:
cd desafio-dio-3

3. Execute o arquivo JavaScript:
node index.js

---

## ⚙️ Como funciona

O usuário informa o nome do herói e sua quantidade de experiência (XP).

Com base nesse valor, o sistema classifica o nível do herói seguindo regras pré-definidas.

### 🏆 Exemplo de saída

O Herói de nome João está no nível Ouro
